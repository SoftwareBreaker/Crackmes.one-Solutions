# Crackmes.one-Solutions

🔓 Solving Crackmes with IDA Pro

# Description / Описание

> **EN**: A collection of crackme analyses and solutions.
> Each solution includes reverse engineering breakdown, algorithm analysis, and bypass/cracking method.
>
> **RU**: Коллекция разборов и решений crackme-задач.
> Каждое решение содержит подробный разбор, анализ алгоритма и способ обхода или взлома.

# 📂 Project Structure / Структура проекта

> **EN** Each challenge is located in its own directory named diff-x (where x is difficult) and follows this structure:

**diff-x/** - Main folder of the challenge.

**[original_file]** - The unmodified, original crackme binary.

**new-[original_file]** - The patched version of the binary (e.g., with bypassed checks, modified jumps, etc.).

**README.md** (or writeup) - Detailed solution, key analysis steps, and algorithms.

> **RU** Каждое задание находится в отдельной папке diff-x (где x сложность) и имеет следующую структуру:

**diff-x/** - Основная папка задания.

**[исходный_файл]** - Оригинальный, нетронутый бинарник crackme.

**new-[исходный_файл]** - Пропатченная версия бинарника (например, с обойденными проверками или измененными инструкциями переходов).

**README.md** (или writeup) - Подробный разбор решения, ключевые этапы анализа и описание алгоритма.
